# RQ2: Review Engagement

Owner: Gabriel Taves

RQ2 studies how reviewers engage with AI-authored pull requests: whether PRs receive review coverage, how quickly review arrives, how much scrutiny occurs, and whether approval is low-friction.

## Scripts

- `scripts/analysis_pipeline.py`: end-to-end review-engagement pipeline.

## Results

The committed `results/` folder includes the generated model tables and figures from the submitted pipeline.

## Data

Place required AIDev parquet files in `RQ2/data/` before rerunning the pipeline.
